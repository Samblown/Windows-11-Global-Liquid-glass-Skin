# Changelog

All notable changes to LiquidGlass will be documented here.

---

## Version 1.0.0

Initial release.

### Features

- Universal acrylic blur effect
- Dark liquid glass tint
- System-wide window support
- Adjustable tint opacity
- Windhawk compatibility

### Notes

- Uses Windows undocumented composition APIs.
- Some applications may not fully support acrylic effects.
